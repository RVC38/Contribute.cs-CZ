# <a name="zone-invalid"></a>zone-invalid

## <a name="warning"></a>Upozornění

`Invalid zone: No "::: zone-end" found. Blocks should be explicitly closed.`

## <a name="resolution"></a>Řešení

Za vaši zónu přidejte `::: zone-end`.
